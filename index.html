Backend Architecture (Python FastAPI)

→POST /api/optimize-route — PyTorch RL model returns optimal waypoints + fuel estimate
→GET /api/hazards — Simulated wind/thruster/Venturi hazard states with FALCON response times
→WS /ws/telemetry — 10Hz stream: position, heading, fuel burn, safe-zone %, recovery timer
→GET /api/datasets/opensky — Proxied ADS-B data with CORS headers for frontend

Rust Microservice (Edge Navigation Layer)

→Handles real-time coordinate transformation at <1ms latency — "memory-safe avionics layer"
→Exposes :9001/nav/compute — takes GPS+INS+Quantum inputs, outputs fused position
→Pitch: "We chose Rust for the nav core — zero undefined behaviour, no GC pauses in safety-critical path"
